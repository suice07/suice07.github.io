# Suice Blog

My personal blog built with [GitHub Pages](https://pages.github.com/) & [Jekyll](http://jekyllrb.com/).

> **Live site**: [suice07.github.io](http://suice07.github.io)

## Quick Start

```bash
# Install dependencies
npm install

# Local preview
jekyll serve -w
```
